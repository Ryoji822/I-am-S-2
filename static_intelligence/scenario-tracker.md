# シナリオ分析 — 4つの未来のどれに向かっているか

> 最終更新: 2026-03-09

**AI市場が今後どう展開するかを、2つの軸（技術の開放度 × 性能の格差）で4つのシナリオに分けて追跡している。現在最も確からしいのは「技術は開くが勝者は出る」（SCN-002, 42%）。GPT-5.4 ProのARC-AGI-2 83.3%（人間72.4%超え）がフロンティア格差拡大を決定的に裏付けた [INFO-015](../Information/2026-03-08/collected-raw.md#INFO-015)。AAIF設立（2025年12月）でプロトコル標準化は実質完了し「開放」方向が確定。「安いAIは誰でも使えるが、最先端は一部企業に集中」する構造がさらに鮮明に。**

**【重要な構造変化: 政府介入の質的変化】** 2026-02-27のAnthropic SCR指定・連邦使用禁止・OpenAI同日DoW契約は、従来の「規制」フレームを超える質的変化を示した。安全性堅持企業への経済的報復→順応企業への利益という「インセンティブ構造の歪み」が顕在化 ([IND-023](../config/indicators.json), **high**昇格)。**

## 2つの軸の意味

4つのシナリオは、2つの根本的な問いの組み合わせで作られている。

**X軸: AIは囲い込まれるか、開放されるか？**
プロトコル層はAAIF（MCP+A2A、Linux Foundation傘下）で標準化が完了。しかし実行環境（Shell/Sandbox）、データ/コンテキスト、ガバナンスの上位レイヤーでは囲い込みが進行中。「閉鎖」は上位レイヤーでの囲い込みを意味する。

**Y軸: トップ企業と後発の性能差は広がるか、縮まるか？**
GPT-5.4 ProのARC-AGI-2 83.3%（人間72.4%超え）が格差拡大の決定的証拠。Gemini 3.1 Pro 77.1%を6.2pt引き離す。一方ByteDance Seed 2.0のコスト1/10はTier 2の追い上げを示す。格差拡大が優勢。

## 各シナリオの詳細

### SCN-001 囲い込みの勝者（23%）— 閉鎖 × 格差拡大

**どういう世界か:** 1-2社が技術力で圧倒しつつ、実行環境・データ・ガバナンスで囲い込む。プロトコル層は開放だが上位レイヤーで移行不能になる。

**今ある証拠:** OpenAIのSkills/Shell/CompactionはMCP上に構築された独自囲い込みレイヤー [INFO-012](../Information/2026-02-28/collected-raw.md#INFO-012)。ただしAWS提携（Bedrockへのstateful runtime展開） [INFO-039](../Information/2026-03-07/collected-raw.md#INFO-039) やWorkspace CLI MCP統合 [INFO-025](../Information/2026-03-08/collected-raw.md#INFO-025) は開放的側面を示し、完全囲い込みと矛盾。Claude CodeがCodexを上回る評価 [INFO-018](../Information/2026-02-28/collected-raw.md#INFO-018) もOpenAI単独支配と矛盾。Nvidia撤退 [INFO-038](../Information/2026-03-07/collected-raw.md#INFO-038) は囲い込み抑制。

**確率が上がる条件:** Skills/Shellが開発者に定着、エンタープライズ顧客のベンダー集中が進む、実行環境の標準化が進まない。
**確率が下がる条件:** 実行環境の標準化が進む、反トラスト規制介入、Seed 2.0級の価格破壊でロックイン解消。

**監視指標:** [IND-001](../config/indicators.json), [IND-002](../config/indicators.json), [IND-003](../config/indicators.json), [IND-015](../config/indicators.json), [IND-018](../config/indicators.json), [IND-019](../config/indicators.json), [IND-020](../config/indicators.json), [IND-024](../config/indicators.json)

### SCN-002 技術は開くが勝者は出る（42%）— 開放 × 格差拡大 ★現在最有力★

**どういう世界か:** AAIF（MCP+A2A）で相互運用性は確保。AIエージェントはツールと自由に接続でき、ベンダー切り替えも技術的に可能。しかしフロンティア性能はOpenAI/Google/Anthropicに集中し「最先端が必要なら特定企業に頼むしかない」状態。

**なぜ今これが最有力か:**
- AAIF標準化完了（2025年12月）: MCP+A2AがLinux Foundation傘下に → **開放**確定
- MCP SDK月間9,700万DL、アクティブサーバー1万以上 → 標準化進展を支持
- **GPT-5.4 Pro ARC-AGI-2 83.3%で人間超え** [INFO-015](../Information/2026-03-08/collected-raw.md#INFO-015)。Gemini 3.1 Pro 77.1%を6.2pt引き離す → **格差拡大**が決定的
- Claude Code愛用率46%首位 [INFO-013](../Information/2026-03-08/collected-raw.md#INFO-013) → 開発者ツール市場でも特定企業への集中
- 価格下落は開放を支持: 年10倍ペースで安いAIは誰でも使える
- 規制（EU AI法2026年8月完全施行）が参入障壁として機能し、小規模プレイヤーを排除

つまり「安いAIは自由に使えるが、最高のAIは一部の企業しか作れない。そして規制が参入障壁を高める」という構造。

**確率が上がる条件:** AAIF標準の実効的採用率が高い、フロンティア性能格差が維持、規制コストが新規参入を阻害。
**確率が下がる条件:** OpenAI Skills/Shellが実行環境で支配的に（→SCN-001）、Seed 2.0級の価格破壊+品質検証でフロンティア追いつき（→SCN-004）。

**監視指標:** [IND-004](../config/indicators.json), [IND-006](../config/indicators.json), [IND-011](../config/indicators.json), [IND-018](../config/indicators.json), [IND-023](../config/indicators.json)

### SCN-003 静かな囲い込み（17%）— 閉鎖 × 収斂

**どういう世界か:** モデル性能差は小さくなるが、データ/ワークフロー/エコシステムのスイッチングコストで顧客が固定化。性能では差がつかないため、データとエコシステムの深さが主戦場。

**今ある証拠:** GPT-5.4 ProのARC-AGI-2 83.3%は格差拡大を示し、収斂前提と矛盾。エンタープライズ3社で88%シェア（[IND-008](../config/indicators.json)）は継続するが、これは囲い込みよりも性能格差の結果と解釈すべき。プロトコル標準化（AAIF）によりプロトコル層でのロックインは困難。

**監視指標:** [IND-007](../config/indicators.json), [IND-008](../config/indicators.json), [IND-009](../config/indicators.json), [IND-011](../config/indicators.json), [IND-015](../config/indicators.json), [IND-017](../config/indicators.json), [IND-019](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-024](../config/indicators.json)

### SCN-004 誰でもAI（18%）— 開放 × 収斂

**どういう世界か:** 性能差がなくなりオープン標準で自由に行き来できる。差別化は業種・機能特化とコスト効率に集中。AIの民主化が最も進んだ世界。

**今ある証拠:** ByteDance Seed 2.0がコスト1/10で性能追い上げ、AAIF標準化完了。ただしGPT-5.4 Pro 83.3%はフロンティア格差がさらに拡大していることを示し、「誰でも同じ」にはまだ遠い。規制コスト（EU AI法）も小規模参入者に不利。

**監視指標:** [IND-004](../config/indicators.json), [IND-011](../config/indicators.json), [IND-017](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-023](../config/indicators.json)

## ブラックスワン（低確率・高影響）

### SCN-BS-001 AI安全性大事故（6%）

AIエージェントの暴走やフェイク情報による社会混乱が起き、厳格な規制が導入されて市場が凍結するシナリオ。Shadow MCP問題で492サーバーが認証欠落 [INFO-014](../Information/2026-02-28/collected-raw.md#INFO-014)。Claude Codeサンドボックス脱出脆弱性 [INFO-031](../Information/2026-03-08/collected-raw.md#INFO-031) も安全性リスクの実例。

### SCN-BS-002 量子コンピューティング×AI融合（3%）

量子コンピュータが突然実用化され、AI訓練・推論のコスト構造が根本から変わるシナリオ。関連する新情報なし。

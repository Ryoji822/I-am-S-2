# シナリオ分析 — 4つの未来のどれに向かっているか

> 最終更新: 2026-02-23

**AI市場が今後どう展開するかを、2つの軸（技術の開放度 × 性能の格差）で4つのシナリオに分けて追跡している。現在最も確からしいのは「技術は開くが勝者は出る」（SCN-002, 33%）。AAIF設立（2025年12月）でプロトコル標準化は実質完了し「開放」方向が確定。しかしGemini 3.1 ProのARC-AGI-2 77.1%に代表されるフロンティア格差は維持されており「格差拡大」も同時進行。プロトコル戦争は終わったが、実行環境・データ・ガバナンスの上位レイヤーで新たな競争が始まっている。**

## 2つの軸の意味

4つのシナリオは、2つの根本的な問いの組み合わせで作られている。

**X軸: AIは囲い込まれるか、開放されるか？**
プロトコル層はAAIF（MCP+A2A、Linux Foundation傘下）で標準化が完了。しかし実行環境（Shell/Sandbox）、データ/コンテキスト、ガバナンスの上位レイヤーでは囲い込みが進行中。「閉鎖」は上位レイヤーでの囲い込みを意味する。

**Y軸: トップ企業と後発の性能差は広がるか、縮まるか？**
Gemini 3.1 ProのARC-AGI-2 77.1%（前世代比146%向上）はフロンティア格差拡大を示す。一方ByteDance Seed 2.0のSWE-Bench 76.5%（自己報告）はTier 2の追い上げを示す。両方が同時進行中。

## 各シナリオの詳細

### SCN-001 囲い込みの勝者（22%）— 閉鎖 × 格差拡大

**どういう世界か:** 1-2社が技術力で圧倒しつつ、実行環境・データ・ガバナンスで囲い込む。プロトコル層は開放だが上位レイヤーで移行不能になる。

**今ある証拠:** OpenAIのSkills/Shell/CompactionはMCP上に構築された独自囲い込みレイヤー。OpenClawマーケットプレイス（700スキル）、Assistants API 2026年8月廃止→Responses APIへの強制移行。$100B+調達。ただしプロトコル層のAAIF標準化により「完全囲い込み」の難度は上昇。

**確率が上がる条件:** Skills/Shellが開発者に定着、エンタープライズ顧客のベンダー集中が進む、実行環境の標準化が進まない。
**確率が下がる条件:** 実行環境の標準化が進む、反トラスト規制介入、Seed 2.0級の価格破壊でロックイン解消。

**監視指標:** [IND-001](../config/indicators.json), [IND-002](../config/indicators.json), [IND-003](../config/indicators.json), [IND-015](../config/indicators.json), [IND-018](../config/indicators.json), [IND-019](../config/indicators.json), [IND-020](../config/indicators.json), [IND-024](../config/indicators.json)

### SCN-002 技術は開くが勝者は出る（33%）— 開放 × 格差拡大 ★現在最有力★

**どういう世界か:** AAIF（MCP+A2A）で相互運用性は確保。AIエージェントはツールと自由に接続でき、ベンダー切り替えも技術的に可能。しかしフロンティア性能はGoogle/OpenAI/Anthropicに集中し「最先端が必要なら特定企業に頼むしかない」状態。

**なぜ今これが最有力か:**
- AAIF標準化完了（2025年12月）: MCP+A2AがLinux Foundation傘下に。OpenAI/Google/Anthropic/Microsoft/AWSが共同創設 → **開放**確定
- フロンティア格差維持: Gemini 3.1 Pro ARC-AGI-2 77.1%、Claude Opus 4.6、GPT-5.2の三つ巴 → **格差拡大**維持
- 価格下落は開放を支持: 年10倍ペースで安いAIは誰でも使える
- 規制（EU AI法2026年8月完全施行）が参入障壁として機能し、小規模プレイヤーを排除

つまり「安いAIは自由に使えるが、最高のAIは一部の企業しか作れない。そして規制が参入障壁を高める」という構造。

**確率が上がる条件:** AAIF標準の実効的採用率が高い、フロンティア性能格差が維持、規制コストが新規参入を阻害。
**確率が下がる条件:** OpenAI Skills/Shellが実行環境で支配的に（→SCN-001）、Seed 2.0級の価格破壊+品質検証でフロンティア追いつき（→SCN-004）。

**監視指標:** [IND-004](../config/indicators.json), [IND-006](../config/indicators.json), [IND-011](../config/indicators.json), [IND-018](../config/indicators.json), [IND-023](../config/indicators.json)

### SCN-003 静かな囲い込み（25%）— 閉鎖 × 収斂

**どういう世界か:** モデル性能差は小さくなるが、データ/ワークフロー/エコシステムのスイッチングコストで顧客が固定化。Googleの検索/Gmail/DriveデータロックインやOpenAI Skills/Shellでのワークフロー固定化。性能では差がつかないため、データとエコシステムの深さが主戦場。

**今ある証拠:** Gemini $1.6/M最安は「性能で差がつかないから価格勝負」を示唆。エンタープライズ3社で88%シェア（[IND-008](../config/indicators.json)）。Gartner: 2027年までに35%の国が地域固有AIプラットフォームにロックイン（データ主権要因）。

**しかしプロトコル標準化（AAIF）により、プロトコル層でのロックインは困難に。** 囲い込みが成立するにはデータ/コンテキスト層（[IND-017](../config/indicators.json)）での優位が必要。Google（検索/Gmail/Drive）、xAI（X+SpaceX衛星データ）、ByteDance（TikTok/Douyin）が独占データを持つ。

**監視指標:** [IND-007](../config/indicators.json), [IND-008](../config/indicators.json), [IND-009](../config/indicators.json), [IND-011](../config/indicators.json), [IND-015](../config/indicators.json), [IND-017](../config/indicators.json), [IND-019](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-024](../config/indicators.json)

### SCN-004 誰でもAI（20%）— 開放 × 収斂

**どういう世界か:** 性能差がなくなりオープン標準で自由に行き来できる。差別化は業種・機能特化とコスト効率に集中。AIの民主化が最も進んだ世界。

**今ある証拠:** OSS台頭（Llama 4 85.5%、GLM5）、Seed 2.0が$0.47/Mで性能追い上げ（自己報告）、AAIF標準化完了。ただしGemini 3.1 Pro ARC-AGI-2 77.1%はフロンティア格差拡大を示し、「誰でも同じ」にはまだ遠い。規制コスト（EU AI法）も小規模参入者に不利。

**監視指標:** [IND-004](../config/indicators.json), [IND-011](../config/indicators.json), [IND-017](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-023](../config/indicators.json)

## ブラックスワン（低確率・高影響）

### SCN-BS-001 AI安全性大事故（6%）

AIエージェントの暴走やフェイク情報による社会混乱が起き、厳格な規制が導入されて市場が凍結するシナリオ。MCP実装の43%にコマンドインジェクション脆弱性、OpenClaw 12%汚染率。EU AI法完全施行（2026年8月）が予防的抑止力として機能する可能性もある。

### SCN-BS-002 量子コンピューティング×AI融合（3%）

量子コンピュータが突然実用化され、AI訓練・推論のコスト構造が根本から変わるシナリオ。関連する新情報なし。

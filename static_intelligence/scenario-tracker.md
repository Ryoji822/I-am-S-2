# シナリオ分析 — 4つの未来のどれに向かっているか

> 最終更新: 2026-03-29

AI市場の展開を、2つの軸（技術の開放度 × 性能の格差）で4つのシナリオに分けて追跡している。

最有力は引き続き「技術は開くが勝者は出る」（SCN-002、39%）。プロトコル標準化は進み、フロンティア性能は3社に集中している。この構造は変わらない。

変わったのは2つ。「囲い込みの勝者」（SCN-001）が20%→22%に上昇した。OpenAI $120B調達、xAI-SpaceX $1.25兆統合で集中化が加速している。もう一つ、「静かな囲い込み」（SCN-003）が21%→22%に微増。エコシステム依存の深化が継続している。

## 2つの軸の意味

4つのシナリオは、2つの根本的な問いの組み合わせで作られている。

**X軸: AIは囲い込まれるか、開放されるか？**

プロトコル層はAAIF（MCP+A2A、Linux Foundation傘下）で標準化が完了した。しかしその上のレイヤー——実行環境（OpenAI Shell、Claude Code Sandbox）、データ/コンテキスト（Frontier Compaction）、ガバナンス（政府承認）——では囲い込みが進行している。「閉鎖」とは、この上位レイヤーでの囲い込みを指す。

SenateのTier 2承認（ChatGPT/Gemini/Copilot承認、Claude除外）は新しいタイプの囲い込みの萌芽。政府が特定ツールを承認することで、エンタープライズのベンダー選択が実質的に制約される。

**Y軸: トップ企業と後発の性能差は広がるか、縮まるか？**

GPT-5.4 ProのARC-AGI-2 83.3%（人間72.4%超え）とo3の87.5%が格差拡大の証拠。GPQA DiamondではGoogle 94.3%が首位。SWE-benchではAnthropic 80.9%が首位。ベンチマークごとにリーダーが違うが、Tier 1の3社（OpenAI/Google/Anthropic）がすべての首位を占めている構造は変わらない。

## 各シナリオの詳細

### SCN-001 囲い込みの勝者（22%）— 閉鎖 × 格差拡大 ★上昇★

1-2社が技術力で圧倒しつつ、実行環境・データ・ガバナンスで囲い込む世界。

20%→22%に上昇した理由は明確。OpenAI $120B調達完了（評価額$730B）[INFO-006](../Information/2026-03-29/collected-raw.md#INFO-006)、xAI-SpaceX $1.25兆統合で資金集中が加速している。上位3社（OpenAI/Anthropic/xAI）で$170B超が集中。

ただし5社が同時にAgent SDKをリリースし（OpenAI Apps SDK、Claude Agent SDK v0.2.86、Gemini API Skills、Grok 4.20、Feishu Aily）、競合が激化している点は反証。NVIDIA NemoClawのオープンソース実行環境も囲い込みの対抗勢力。AWS提携でOpenAIがBedrockに展開したことも、単一企業の囲い込みとは矛盾する [INFO-039](../Information/2026-03-07/collected-raw.md#INFO-039)。

確率が上がる条件: Skills/Shellが開発者に定着、エンタープライズのベンダー集中が進む。
確率が下がる条件: 実行環境の標準化が進む（NVIDIA NemoClaw）、反トラスト規制介入。

### SCN-002 技術は開くが勝者は出る（39%）— 開放 × 格差拡大 ★現在最有力★

AAIF標準で相互運用性は確保されるが、フロンティア性能はTier 1の3社に集中する世界。

引き続き最有力だが、41%→39%に微減。微減の理由は、SCN-001へのシフト——資金集中が加速し、囲い込み圧力が強まっている。開放と囲い込みが同時に進むため、このシナリオとSCN-001/SCN-003の境界がぼやけてきた。

このシナリオを支持する証拠は依然として最も厚い。MCP SDK月間9,700万DL。193カ国のジュネーブAI合意。EU AI法2026年8月完全施行。年10倍ペースの価格下落。これらすべてが「安いAIは自由に使えるが、最高のAIは一部の企業しか作れない」構造を補強する。

7社同時SDK更新も「開放」側の証拠——各社が独自実装しつつ、プロトコル層ではMCP準拠を維持。

確率が上がる条件: AAIF標準の実効的採用率が高い、フロンティア性能格差が維持、規制コストが新規参入を阻害。
確率が下がる条件: 上位レイヤーの囲い込みが支配的に（→SCN-001/SCN-003）、価格破壊+品質検証でフロンティア追いつき（→SCN-004）。

監視指標: [IND-004](../config/indicators.json), [IND-006](../config/indicators.json), [IND-011](../config/indicators.json), [IND-018](../config/indicators.json), [IND-023](../config/indicators.json)

### SCN-003 静かな囲い込み（22%）— 閉鎖 × 収斂

モデル性能差は小さくなるが、データ・ワークフロー・エコシステムのスイッチングコストで顧客が固定化する世界。

21%→22%に+1pt上昇。3つの証拠が積み重なっている。

1つ目、SenateのTier 2承認。ChatGPT/Gemini/Copilotを承認しClaudeを除外した [INFO-008](../Information/2026-03-20/collected-raw.md#INFO-008)。性能差ではなく政治的判断でベンダーが選別される新しい形のロックイン。企業が連邦政府と取引するなら、承認済みツールを使わざるを得ない。

2つ目、エージェントSDK乱立。7社同時リリースは「開放」に見えるが、各社のSDKは相互運用しない。エージェントを一度構築すると、SDK固有のAPIに依存し、切り替えコストが発生する。

3つ目、データ囲い込みの加速。xAI Grok 4.20のx_search内蔵 [INFO-016](../Information/2026-03-29/collected-raw.md#INFO-016)、Google Personal Intelligence無料化（Gmail/Photos/YouTube統合）[INFO-007](../Information/2026-03-29/collected-raw.md#INFO-007) でデータ独占活用が明確化。

確率が上がる条件: Agent SDK間の非互換性が固定化、政府承認がベンダー選択を実質的に制約、エコシステム依存の深化。
確率が下がる条件: Agent SDKの標準化が進む、反トラスト規制介入、ベンダー切り替えツールの出現。

監視指標: [IND-007](../config/indicators.json), [IND-008](../config/indicators.json), [IND-009](../config/indicators.json), [IND-011](../config/indicators.json), [IND-015](../config/indicators.json), [IND-017](../config/indicators.json), [IND-019](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-024](../config/indicators.json)

### SCN-004 誰でもAI（17%）— 開放 × 収斂

性能差がなくなりオープン標準で自由に行き来できる世界。

18%→17%に-1pt低下。ByteDance Seed 2.0のコスト1/10、DeerFlow 2.0のオープンソース化（39,000+ stars）[INFO-018](../Information/2026-03-29/collected-raw.md#INFO-018)、NVIDIA NemoClawのオープンソース実行環境 [INFO-036](../Information/2026-03-25/collected-raw.md#INFO-036) は収斂方向の証拠だが、GPT-5.4 Pro 83.3%やGPQA Diamond 94.3%が示すフロンティア格差は依然として大きい。「誰でも同じ」にはまだ距離がある。

監視指標: [IND-004](../config/indicators.json), [IND-011](../config/indicators.json), [IND-017](../config/indicators.json), [IND-020](../config/indicators.json), [IND-022](../config/indicators.json), [IND-023](../config/indicators.json)

## ブラックスワン（低確率・高影響）

### SCN-BS-001 AI安全性大事故（11%）

AIエージェントの暴走やセキュリティ事故で社会的混乱が起き、厳格な規制が導入されて市場が凍結するシナリオ。

11%で変動なし。具体的インシデントの蓄積が継続している。OpenClaw Skills Registryで135,000超のインスタンスが認証なしで露出（63%が脆弱）[INFO-050](../Information/2026-03-29/collected-raw.md#INFO-050)。MetaでSev-1インシデント（AIエージェントが権限外アクセス誘発）。MCP Shadow ITではセキュリティレビューなしに本番環境にMCPサーバーが接続されている。HiddenLayer調査: AI侵害の8件に1件以上がエージェントシステム。

個々のインシデントは修正済みだが、パターンが見える。エージェントの自律性が上がるほど攻撃面が広がり、プロトコル標準化がセキュリティレビューより速く進んでいる。「大事故」の確率が上がっているのは、単一の大きなリスクではなく、小さなリスクの同時多発。

### SCN-BS-002 量子コンピューティング×AI融合（3%）

量子コンピュータが実用化され、AI訓練・推論のコスト構造が根本から変わるシナリオ。関連する新情報なし。

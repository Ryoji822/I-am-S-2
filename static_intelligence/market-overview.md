# AI市場の現在地
> 最終更新: 2026-05-06

安いAIは誰でも使える。だが最先端を作れるのは3社だけ——市場はその構造に向かっている（SCN-002、38%）。だが「静かな围い込み」（SCN-003、28%）が3ラウンド連続で確率を上げている。围い込みシグナルの構造的蓄積が進んでいる。

2026年5月第一週、市場構造を変える出来事が重なった。**Microsoft Agent 365がGA化**し [INFO-026](../Information/2026-05-06/collected-raw.md#INFO-026)、エンタープライズエージェントガバナンスの新段階に入った。**AnthropicとOpenAIが同週にJVを設立**し [INFO-056](../Information/2026-05-06/collected-raw.md#INFO-056)、Palantir型FDEモデルを同時採用した。围い込みに「金融次元」が加わった。**BenchLM総合**でClaude Mythos Previewが99、Gemini 3.1 Proが93、GPT-5.4 Proが92——上位3社の差は6-7ptに縮まり [INFO-028](../Information/2026-05-06/collected-raw.md#INFO-028)、「勝者」格差として限界的になっている。

## 今、市場で何が起きているか

### JV/FDEモデルの同時採用 — エンタープライズGo-to-Marketが変わった

Anthropic（Blackstone/H&F/GS $1.5B評価額JV）とOpenAI（TPG/Brookfield $10B評価額JV「The Development Company」）が同週にJVを設立した [INFO-056](../Information/2026-05-06/collected-raw.md#INFO-056)。双方がPalantir型FDE（Forward-Deployed Engineer）モデルを採用。PEファンドのポートフォリオ企業への優先販売アクセスが围い込みの新しい「金融次元」になった。API提供からJV+埋め込みエンジニアへの構造的深化。

### Agent Platform三社鼎立 — 均質化が進む

Microsoft Agent 365 GA [INFO-026](../Information/2026-05-06/collected-raw.md#INFO-026)、Google Gemini Enterprise Agent Platform [INFO-007](../Information/2026-05-06/collected-raw.md#INFO-007)、AWS Bedrock AgentCore [INFO-025](../Information/2026-05-06/collected-raw.md#INFO-025) が全て稼働。Agent Platformが三社鼎立になり、均質化が進む。BenchLM上位3社の差は6-7ptで「勝者」格差として限界的 [INFO-028](../Information/2026-05-06/collected-raw.md#INFO-028)。SCN-002の「勝者」前提を侵食する方向。

### 围い込みシグナルの構造的蓄積

SCN-003が3ラウンド連続で確率上昇（25→28%）。MSFT Agent 365 GA [INFO-026](../Information/2026-05-06/collected-raw.md#INFO-026) + Copilot credit system [INFO-050](../Information/2026-05-06/collected-raw.md#INFO-050) + ベンダーロックイン深化 [INFO-033](../Information/2026-05-06/collected-raw.md#INFO-033) + JV围い込み（金融次元）[INFO-056](../Information/2026-05-06/collected-raw.md#INFO-056) の4重C。性能差が縮まる中で、エコシステムのスイッチングコストで围い込む構造が強まっている。

### エンタープライズ採用の臨界点に接近

D&B 10,000社調査で97%がAI活動を報告、30%が本番スケールへ移行 [INFO-034](../Information/2026-05-06/collected-raw.md#INFO-034)。パイロット→本番移行率はQ3 2025の11%からQ1 2026の18%へ上昇（過去最大）[INFO-035](../Information/2026-05-06/collected-raw.md#INFO-035)。GenAI早期採用者の92%がポジティブROI [INFO-039](../Information/2026-05-06/collected-raw.md#INFO-039)。ServiceNowはITサービスデスクで人間比99%高速解決 [INFO-038](../Information/2026-05-06/collected-raw.md#INFO-038)。

だが24.4%の組織のみがAIエージェントへの完全な可視性を持つ [INFO-036](../Information/2026-05-06/collected-raw.md#INFO-036)。AIエージェントが本番データベースを削除するインシデントも発生 [INFO-009](../Information/2026-05-06/collected-raw.md#INFO-009)。採用は加速しているが、安全可視性が追いついていない。

### 「書く」から「評価する」への大転換が加速

プログラマー雇用が2023-2025年で27.5%減少 [INFO-058](../Information/2026-05-06/collected-raw.md#INFO-058)。GenAI採用企業のジュニア採用22%削減 [INFO-062](../Information/2026-05-06/collected-raw.md#INFO-062)。初期キャリア雇用はAI露出の高い職業で16%減少 [INFO-057](../Information/2026-05-06/collected-raw.md#INFO-057)。Salesforce CEOがAIエージェントで約4000のCS職削減を確認 [INFO-077](../Information/2026-05-06/collected-raw.md#INFO-077)。2023年1月〜2026年4月で約76万人のテック労働者がレイオフ [INFO-059](../Information/2026-05-06/collected-raw.md#INFO-059)。

逆説も並存する。Klarnaが700人AI代替から人間再雇用に転じた [INFO-044](../Information/2026-05-06/collected-raw.md#INFO-044)。AI投資の95%がリターンなしという指摘も [INFO-045](../Information/2026-05-06/collected-raw.md#INFO-045)。（[IND-022](../config/indicators.json)、high）

### 価格二極化の深化

GPT-5.5は$5/$30（前世代2倍）。DeepSeek V4は$1.74/$3.48 per M tokens [INFO-028](../Information/2026-05-06/collected-raw.md#INFO-028)。xAI Grok 4.3は$1.25/$2.50。Codexがper-token価格に移行 [INFO-048](../Information/2026-05-06/collected-raw.md#INFO-048)。AIコストは一部企業で単月50%急増 [INFO-049](../Information/2026-05-06/collected-raw.md#INFO-049)。「安いAI」の時代が終わりつつあるという指摘も [INFO-067](../Information/2026-05-06/collected-raw.md#INFO-067)。

### OSS vs 商用のギャップが急速に縮小

Stanford 2026 AI Indexによると、オープンソースAIとフロンティアモデルの性能ギャップが約8%から約1.7%に縮小 [INFO-051](../Information/2026-05-06/collected-raw.md#INFO-051)。LLMのコモディティ化の兆候。だしBenchLM上位12pt差は残る [INFO-028](../Information/2026-05-06/collected-raw.md#INFO-028)。

### インフラ過熱と資本集中

3社合算CapEx ~$560Bに向かって推移 [INFO-042](../Information/2026-05-06/collected-raw.md#INFO-042)。Anthropic $900B評価額協議 [INFO-052](../Information/2026-05-06/collected-raw.md#INFO-052)。Sierra $950M Series E [INFO-054](../Information/2026-05-06/collected-raw.md#INFO-054)。Amazon Anthropic投資$80億→$700億以上 [INFO-053](../Information/2026-05-06/collected-raw.md#INFO-053)。（[IND-029](../config/indicators.json)、high）

### EU AI法執行が確定

EU AI Act高リスク期限が2026年8月2日に確定。提案された猶予は否決された [INFO-040](../Information/2026-05-06/collected-raw.md#INFO-040) [INFO-041](../Information/2026-05-06/collected-raw.md#INFO-041)。

## 寡占と分散のどちらに向かうか

**集中に向かう力**: Agent Platform三社鼎立。JV围い込み（金融次元）。Copilot credit system。Anthropic $900B評価額協議。3社合算CapEx ~$560B。

**分散に向かう力**: OSSギャップ8%→1.7% [INFO-051](../Information/2026-05-06/collected-raw.md#INFO-051)。MCP全社サポート [INFO-015](../Information/2026-05-06/collected-raw.md#INFO-015)。Red Hat MCP Gateway [INFO-016](../Information/2026-05-06/collected-raw.md#INFO-016)。BenchLM上位差6-7ptで均質化。

現時点では「技術は開くが、性能トップは集中する」（SCN-002、38%）が最有力だが、「静かな围い込み」（SCN-003、28%）が3ラウンド連続で迫っている。詳細は `scenario-tracker.md` を参照。

# Anthropic インテリジェンス・プロファイル

> 最終更新: 2026-04-28
> 確度: 高

Googleが最大$40Bを投じ、Anthropicの評価額は$350Bに跳ね上がった [INFO-054](../Information/2026-04-28/collected-raw.md#INFO-054)。即時$10B、条件達成で追加$30B。Amazonも数日前に$5Bを投じている。Anthropicが集めた資金は「ほとんどの国の外貨準備を上回る」規模になった。

だが逆説は三重から四重になった。安全性を貫いて政府に排除された企業が、排除されたことで民間の信頼を獲得し、信頼が収益を押し上げ、収益がさらに資金を呼ぶ。**$30B ARR到達がNYTで確認された** [INFO-075](../Information/2026-04-28/collected-raw.md#INFO-075)。OpenAIの$25Bを初めて逆転した。ただし$22B説も依然として併記される [INFO-046](../Information/2026-04-24/collected-raw.md#INFO-046)。

そして法的状況が動いた。**連邦裁判所がSCR指定の仮差し止めを認可**した [INFO-043](../Information/2026-04-28/collected-raw.md#INFO-043)。DC回路裁判所は政府の控訴延期請求を拒否。政府排除は確定していなかった。

## 戦略方向性

### 安全性でエンタープライスを取る（H-ANT-001、確度52%）

SOC2準拠、Compliance API、ASL-3保護で規制業界を取る戦略。エンタープライズ40%シェアとRampデータ（新規70%がAnthropic選択）は、民間では機能している証拠。EU AI法完全施行（2026年8月）は追い風。

**Google $40B投資はこの仮説に複雑な影響を与える**。Anthropicは「ほとんどの国の外貨準備を上回る」資金を得た——安全性研究への投資余力は飛躍的に増した。同時に、Googleの14%持ち分はAnthropicの独立性に疑問を生む。安全性堅持がGoogleの商業利益と衝突したとき、どうなるか。

SCR指定の**仮差し止め**は政府排除の不確実性を高めた——Anthropicの法的地位は「排除確定」から「係争中」に変わった [INFO-043](../Information/2026-04-28/collected-raw.md#INFO-043)。$30B NYT確認は安全性が民間市場で報われている証拠 [INFO-075](../Information/2026-04-28/collected-raw.md#INFO-075)。ただし$22B説が正しければ成長の見え方が変わる。

確度は52%で維持 [Arbiter v3.62](../state/arbiter-2026-04-27.md)。

### 開発者ツールで差別化する（H-ANT-002、確度67%）— 全仮説中最高

Claude Code + Agent SDKの3点セットによる差別化。SWE-bench Verified 80.9%で首位。

**Claude Codeポストモーテム**（A-1品質）は透明性の強いC [INFO-069](../Information/2026-04-24/collected-raw.md#INFO-069)。3つの独立した問題を完全特定し全修正済み。Opus 4.7が自社のバグを発見した点は新モデルの品質改善を実証する。

だが**ProプランからのClaude Code削除** [INFO-047](../Information/2026-04-28/collected-raw.md#INFO-047) とAPI利用料高騰（20日間で$1200の事例）は供給側構造リスクを深めた。Pro→OpenAIへの大量流出も報告されている [INFO-059](../Information/2026-04-28/collected-raw.md#INFO-059)。**Memory for Managed Agentsパブリックベータ** [INFO-078](../Information/2026-04-28/collected-raw.md#INFO-078) は製品面の進展。

確度は67% [Arbiter v3.62](../state/arbiter-2026-04-27.md)。ポストモーテム透明性C vs コンピュート不足I3件が相殺。

### マルチクラウドで広げる（H-ANT-003、確度6%）

Claudeは3大クラウド全てで利用可能な唯一のフロンティアAIモデル。だが確度は6%に低下。Amazon提携拡大（5GW Trainium・$100B AWS投資）でAWS依存が決定的に深まった [INFO-032](../Information/2026-04-22/collected-raw.md#INFO-032)。**Google $40B投資**はGoogle依存も加わる。「マルチクラウド同等機能」要件からの乖離は修復不可能に近い。

### 代替仮説: 政府圧力による萎縮効果（H-GOV-001、確度46%）

Anthropic-Pentagon対立がもたらす業界全体への萎縮効果を追跡中。確度は46% [Arbiter v3.62](../state/arbiter-2026-04-27.md)。SCR指定の仮差し止めは政府圧力の歯止めとなるI。だがOpenAIのPentagon契約獲得・"QuitGPT"ボイコットは「安全性堅持企業が罰され、順応企業が報われる」構造のC。

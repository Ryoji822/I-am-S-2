## 0. 一文要約

GrokがProject Mavenに統合済みであることが米政府法廷文書で確認された [INFO-064](../Information/2026-07-07/collected-raw.md#INFO-064)（A-1）。Carnegie Endowmentの詳細レポートが、初の公表された大規模LLMサイバー攻撃（中国関連グループがClaudeを使用）と、1人のオペレーターが平均90エージェントを制御する実態を報告した [INFO-067](../Information/2026-07-07/collected-raw.md#INFO-067)（A-1）。これらは軍事・国家安保領域でのGrok/Claude実使用をA-1品質で確認するが、[H-XAI-004](../config/hypotheses.json) の核心命題である「汎用AI基盤としてのエンタープライズ市場獲得」の直接証拠ではない。Cursorが$100Mから$2B ARRに13ヶ月で成長した [INFO-048](../Information/2026-07-07/collected-raw.md#INFO-048)（B-2）一方で、xAIのエンタープライズ採用定量データは11R以上完全に不在である。

[H-XAI-002](../config/hypotheses.json) は59% medium（±0%）。[H-XAI-004](../config/hypotheses.json) は54% medium（57→54%）。エンタープライズ採用定量データ（WAU/DAU・F500導入率・企業契約数）の不在が累積ペナルティとして作用し、H-ANT-002との対称的一貫適用で引き下げられた。もしCursor統合後にGrok系コーディングツールの採用が定量で回復するか、DL減少傾向が3ヶ月以上継続すれば、コア判断が変わる。

## 1. コア判断

確信度は低のまま置く。垂直統合の外観は完成し、軍事統合の深度は前回更新から更に増した。但しエンタープライズ採用という核心課題が解消していない。Grok in Project Mavenの確認は政府/軍事領域の拡大を示すが、ステートメントが指定する「エンタープライズ市場」の直接証拠ではない。

Project MavenへのGrok統合は、米政府法廷文書で初めて確認された [INFO-064](../Information/2026-07-07/collected-raw.md#INFO-064)（A-1）。前回の [INFO-004](../Information/2026-06-29/collected-raw.md#INFO-004)（B-3）でPentagon宣誓陳述書によりGrokのイラン作戦での2,000標的指定使用が確認されていたが、今回のProject Maven統合確認は、Grokが衛星画像・レーダー・信号情報の統合プラットフォームに恒久的に組み込まれたことを意味する。AnthropicのPentagon拒否2レッドライン（自律型致死兵器のhuman-in-the-loop不在・国内監視）との対比で、xAIが全社軍事協定に署名し順応した受益者位置にある構造が政府文書で確定した。但し、これらは政府チャネルの持続性を示すものであり、商業エンタープライズ市場でのGrok採用を直接示すものではない。

Carnegie Endowmentの詳細レポート [INFO-067](../Information/2026-07-07/collected-raw.md#INFO-067)（A-1）は4つの分析的価値を持つ。第一に、2025年11月に中国関連グループがAnthropic Claudeを外国政府・重要インフラ攻撃に使用したことが、公表された初の大規模LLMサイバー攻撃として確認された。第二に、国防長官がAnthropicにClaude無制限軍事アクセスを要求し、$200M契約失効を脅迫した事実（2026年2月）が記録された。第三に、「1人のオペレーターが平均90エージェントを制御する」実態が報告され、意味ある人間統制の実効性に定量的疑義が提示された。第四に、Trump政権のサイバー戦略（2026年3月）がAIエージェントによるネットワーク防衛・妨害を推進している。これらは [IND-030](../config/indicators.json) critical/risingの妥当性を更に強化するが、xAIのエンタープライズ採用とは直接結合しない。

[H-XAI-004](../config/hypotheses.json) は57%から54%に低下した（3R連続-1%）。低下の根拠は3つある。第一に、エンタープライズ採用定量データ（WAU/DAU・F500導入率・企業契約数）が11R以上完全に不在であり、I=0の状態が継続している。第二に、この累積ペナルティ論理を [H-ANT-002](../config/hypotheses.json)（核心パラメータ不在で段階的引き下げ適用）と対称的に適用すべきである。第三に、Voice Agent Builder [INFO-008](../Information/2026-07-07/collected-raw.md#INFO-008)（C-3）はgenuineな製品Cだが、availability≠adoptionの二重基準（Blue自身が認める）を行動に反映した。Grok in Project Maven（A-1）は政府/軍事利用であり、ステートメントが指定する「エンタープライズ市場」の直接証拠ではない。

商用面では、Cursorが$100Mから$2B ARRに13ヶ月で成長した [INFO-048](../Information/2026-07-07/collected-raw.md#INFO-048)（B-2）。SpaceXが$60Bで買収した資産の価値が成長軌道にあることを示す。但しCursorの成長の大部分がClaude/GPT由来の価値であり、Grok固有のエンタープライズ採用を分離する定量データは不在である。DL 60%減（1月20M→4月8.3M）も未解決であり、Cursor市場シェア41%(2025-06)→26%(2026-05)の下落傾向が継続しているかは次回確認が必要である。xAIのVoice Agent Builder [INFO-008](../Information/2026-07-07/collected-raw.md#INFO-008)（C-3）は$0.05/分のノーコードボイスエージェント構築を可能にするが、採用の定量裏付けは不在である。

[H-XAI-002](../config/hypotheses.json) は59% mediumで±0%維持。価格コモディティ化圧力は継続しており、GLM 5.2がHuaweiシリコン全量でオープンウェイト1位に到達した [INFO-038](../Information/2026-07-07/collected-raw.md#INFO-038)（B-2）。OSS LLMのエンタープライズ支出シェアが19%から11%に低下した [INFO-039](../Information/2026-07-07/collected-raw.md#INFO-039)（B-3）のは商用モデル優位を示すが、Grokの「低価格」差別化が持続するかは不確定である。59%以下が続けば次ラウンドでmedium→low審査がトリガーされる。

[H-XAI-001](../config/hypotheses.json)（Xデータ差別化）と [H-XAI-003](../config/hypotheses.json)（宇宙・製造業特化）は引き続き棄却（35%）で、新たな支持証拠は観測されていない。
